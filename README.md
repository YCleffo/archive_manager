# Курсовой проект: Менеджер файлов (Архиватор)

**Тема:** Разработка десктопного приложения — файлового менеджера с функциями работы с архивами и поиска.
**Выполнил:** студент группы РИВ-240938у, Никогосян Юрий Сасунович.

---

## Описание проекта

Проект представляет собой оконное приложение на языке Python, объединяющее в себе возможности простого файлового менеджера, архиватора и инструмента для поиска файлов. Приложение позволяет осуществлять навигацию по файловой системе, выполнять базовые операции с файлами и папками, а также создавать и распаковывать архивы различных форматов.

## Используемые технологии

- **Язык программирования:** Python 3.10+
- **Графический интерфейс:** `tkinter`, `ttk` (стандартная библиотека Python)
- **Работа с файловой системой:** `os`, `pathlib`, `shutil`
- **Работа с архивами:** `zipfile` (создание и чтение ZIP), `tarfile` (чтение TAR, GZ, BZ2)
- **Многопоточность:** `threading`, `queue` (для предотвращения зависания интерфейса при долгих операциях поиска и архивации)
- **Сторонние библиотеки:** Не используются (проект полностью базируется на стандартной библиотеке Python).

## Функциональные возможности

1. **Файловый менеджер:**
    - Навигация по файловой системе (переход по директориям, переход на уровень вверх, возврат назад).
    - Выполнение базовых файловых операций: создание папок, переименование, копирование, перемещение и удаление файлов/папок.
    - Подсчёт размера папок.
    - Открытие файлов с помощью системных приложений по умолчанию.

2. **Работа с архивами:**
    - Создание `.zip` архивов из выбранных файлов и папок с сохранением их структуры.
    - Распаковка архивов популярных форматов: `.zip`, `.tar`, `.tar.gz`, `.tgz`, `.tar.bz2`.
    - Безопасная распаковка (защита от уязвимости Zip Slip и путей вида `../../file`).
    - Предварительный просмотр содержимого поддерживаемых архивов без необходимости их полной распаковки.

3. **Поиск файлов:**
    - Поиск по имени файла или папки в текущей директории и поддиректориях.
    - Поиск текста **внутри** файлов (полнотекстовый поиск с ограничениями для бинарных файлов).
    - Фильтрация результатов поиска по конкретным расширениям файлов.
    - Асинхронное выполнение поиска с возможностью прерывания процесса (кнопка "Стоп").

```text
archive_manager/
├── main.py                    # Главная точка входа в приложение

```

## Инструкция по запуску

### Для ОС Windows

Для запуска программы достаточно дважды кликнуть по файлу:

```bat
run.bat
```

Или выполнить команду в консоли:

```bat
python main.py
```

### Для ОС Linux / macOS

Необходимо выдать права на исполнение скрипту и запустить его:

```bash
chmod +x run.sh
./run.sh
```

Или запустить напрямую через интерпретатор Python:

```bash
python3 main.py
```
