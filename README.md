# Курсовой проект: Менеджер файлов (Архиватор)

- **Выполнил:** Никогосян Юрий Сасунович
- **Группа:** РИВ-240938у
- **Тема:** Менеджер файлов (Архиватор)

Нативное десктопное приложение для работы с файлами, папками и архивами. Программа предоставляет удобный графический интерфейс для навигации по файловой системе, выполнения базовых операций с объектами, создания ZIP-архивов, распаковки архивов и поиска файлов.

## Основные возможности

- Просмотр содержимого папок в табличном виде.
- Переход назад, на уровень выше и в домашнюю директорию.
- Переход к папке по введённому пути.
- Выбор папки через системный диалог.
- Создание новых папок.
- Переименование файлов и папок.
- Копирование и перемещение выбранных объектов.
- Удаление файлов и папок без помещения в корзину.
- Открытие файлов в системных приложениях по умолчанию.
- Подсчёт размера выбранной папки.
- Создание ZIP-архивов из выбранных файлов и папок.
- Распаковка архивов `.zip`, `.tar`, `.tar.gz`, `.tgz`, `.tar.bz2`.
- Просмотр содержимого архива без распаковки.
- Поиск файлов и папок по имени.
- Поиск текста внутри поддерживаемых текстовых файлов.
- Фильтрация поиска по расширениям.

## Интерфейс

Приложение построено как нативная оконная программа на Qt. Главный экран состоит из:

- верхней панели быстрых действий;
- строки текущего пути;
- таблицы файлов с сортировкой по столбцам;
- контекстного меню для выбранных объектов;
- скрываемой панели поиска.

Панель поиска открывается кнопкой `Поиск` или сочетанием `Ctrl+F`. Её можно закрыть, остановить текущий поиск или сбросить запрос, фильтры и результаты.

## Горячие клавиши

| Действие | Клавиши |
| --- | --- |
| Назад | `Alt+Left` |
| Вверх | `Alt+Up` |
| Домой | `Alt+Home` |
| Обновить список | `F5` |
| Открыть поиск | `Ctrl+F` |
| Открыть выбранный объект | `Enter` |
| Переименовать выбранный объект | `F2` |
| Удалить выбранные объекты | `Delete` |

## Технологии

- Python 3.10+
- PySide6 / Qt
- pathlib, os, shutil для работы с файловой системой
- zipfile и tarfile для работы с архивами
- QThreadPool и QRunnable для фоновых операций

## Установка

Сначала установите зависимости:

```bash
python -m pip install -r requirements.txt
```

Если в системе команда `python` указывает не на Python 3, используйте:

```bash
python3 -m pip install -r requirements.txt
```

## Запуск

### Windows

Через файл запуска:

```bat
run.bat
```

Или напрямую:

```bat
python main.py
```

### Linux / macOS

Через файл запуска:

```bash
chmod +x run.sh
./run.sh
```

Или напрямую:

```bash
python3 main.py
```

## Поддерживаемые архивы

| Формат | Создание | Распаковка | Просмотр содержимого |
| --- | --- | --- | --- |
| `.zip` | Да | Да | Да |
| `.tar` | Нет | Да | Да |
| `.tar.gz` | Нет | Да | Да |
| `.tgz` | Нет | Да | Да |
| `.tar.bz2` | Нет | Да | Да |

## Примечания

- Удаление выполняется напрямую, без корзины.
- Распаковка архивов выполняется с проверкой путей, чтобы файлы из архива не могли быть записаны за пределы выбранной папки.
- Долгие операции запускаются в фоне, чтобы интерфейс оставался отзывчивым.
