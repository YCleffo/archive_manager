# Курсовой проект: Менеджер файлов (Архиватор)

- **Выполнил:** Никогосян Юрий Сасунович
- **Группа:** РИВ-240938у
- **Тема:** Менеджер файлов (Архиватор)

Нативное десктопное приложение для работы с файлами, папками и архивами. Программа предоставляет удобный графический интерфейс для навигации по файловой системе, выполнения базовых операций с объектами, создания ZIP-архивов, распаковки архивов и поиска файлов.

## Основные возможности

- Просмотр содержимого папок в табличном виде.
- Переход назад, на уровень выше и в домашнюю директорию.
- Переход вперёд и назад по истории открытых папок.
- Переход к папке по введённому пути.
- Выбор папки через внутренний диалог приложения.
- Создание новых папок.
- Переименование файлов и папок.
- Копирование и перемещение выбранных объектов с отображением статуса буфера обмена на кнопке вставки.
- Удаление файлов и папок с перемещением в системную корзину.
- Одинарный клик по объекту показывает правую панель превью: миниатюра изображения читается в уменьшенном виде даже для больших файлов, для видео показывается первый кадр как картинка без запуска проигрывателя.
- Двойной клик по файлу открывает его в стандартной программе ОС, двойной клик по папке открывает папку внутри приложения.
- Подсчёт размера выбранной папки через панель действий или кнопку `Посчитать` в колонке размера.
- Создание ZIP-архивов из выбранных файлов и папок.
- Распаковка архивов `.zip`, `.tar`, `.tar.gz`, `.tgz`, `.tar.bz2`.
- Просмотр содержимого архива без распаковки.
- Поиск файлов и папок по имени.
- Поиск текста внутри поддерживаемых текстовых файлов.
- Фильтрация поиска по расширениям.
- Информативный статус-бар, отображающий количество выбранных файлов и папок.

## Интерфейс

Приложение построено как нативная оконная программа на Qt. Главный экран состоит из:

- верхней панели быстрых действий;
- строки текущего пути;
- таблицы файлов с сортировкой по столбцам;
- контекстного меню для выбранных объектов;
- скрываемой панели поиска;
- правой панели превью выбранного объекта.

Панель поиска открывается кнопкой `Поиск` или сочетанием `Ctrl+F`. Её можно закрыть, остановить текущий поиск или сбросить запрос, фильтры и результаты.

Кнопки мыши `Назад` и `Вперёд`, если они есть на устройстве, используются для перехода по истории папок.

| --- | --- |
| Назад | `Alt+Left` |
| Вперёд | `Alt+Right` |
| Вверх | `Alt+Up` |
| Домой | `Alt+Home` |
| Обновить список | `F5` |

## Технологии

- Python 3.10+
- PySide6 / Qt
- imageio-ffmpeg для получения первого кадра видео в панели превью
- pathlib, os, shutil для работы с файловой системой
- zipfile и tarfile для работы с архивами
- QThreadPool и QRunnable для фоновых операций

## Установка

Сначала установите зависимости:

```bash
python -m pip install -r requirements.txt
```

Если в системе команда `python` указывает не на Python 3, используйте:

```bash
python3 -m pip install -r requirements.txt
```

## Запуск

### Windows

Через файл запуска:

```bat
run.bat
```

Или напрямую:

```bat
python main.py
```

### Linux / macOS

Запуск напрямую:

```bash
python3 main.py
```

## Примечания

- Удаление выполняется с перемещением в системную корзину ОС (через send2trash), обеспечивая безопасность данных.
- `Ctrl+Z` отменяет поддерживаемые файловые операции: создание папки, переименование, копирование, перемещение и создание ZIP-архива. Удаление через `Ctrl+Z` не восстанавливается.
- Распаковка архивов выполняется с проверкой путей, чтобы файлы из архива не могли быть записаны за пределы выбранной папки.
- Долгие операции запускаются в фоне, чтобы интерфейс оставался отзывчивым.
- Превью изображений создаётся через `QImageReader.setScaledSize()`, поэтому приложение не пытается показывать исходник целиком в интерфейсе.
- Превью видео строится как один кадр через `imageio-ffmpeg`; если ffmpeg недоступен, приложение показывает информацию о файле и причину, почему кадр не создан.
