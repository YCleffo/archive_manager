# Курсовой проект: Менеджер файлов (Архиватор)

**Тема:** разработка нативного десктопного приложения - файлового менеджера с функциями работы с архивами и поиска.
**Выполнил:** студент группы РИВ-240938у, Никогосян Юрий Сасунович.

---

## Описание проекта

Проект представляет собой нативное оконное приложение на Python. Оно объединяет возможности файлового менеджера, архиватора и инструмента поиска файлов. Приложение работает напрямую с файловой системой через стандартные Python API и системные диалоги Qt, поэтому сохраняет полный доступ к файлам и папкам без web-оболочки.

Интерфейс перенесён с `tkinter` на `PySide6` / Qt: окно стало современнее, использует нативные элементы управления, единый визуальный стиль, таблицы с сортировкой, контекстное меню и асинхронные фоновые операции для долгих задач.

## Используемые технологии

- **Язык программирования:** Python 3.10+
- **Графический интерфейс:** `PySide6` / Qt
- **Работа с файловой системой:** `os`, `pathlib`, `shutil`
- **Работа с архивами:** `zipfile` (создание и чтение ZIP), `tarfile` (чтение TAR, GZ, BZ2)
- **Фоновые операции:** `QThreadPool`, `QRunnable`, `threading.Event`
- **Поиск:** рекурсивный поиск по имени и, при необходимости, по содержимому текстовых файлов

## Функциональные возможности

1. **Файловый менеджер:**
   - Навигация по файловой системе: назад, вверх, домой, переход по введённому пути, выбор папки через системный диалог.
   - Открытие файлов через приложения по умолчанию.
   - Создание папок, переименование, копирование, перемещение и удаление файлов/папок.
   - Подсчёт размера папок в фоновом режиме.
   - Таблица файлов с едиными монохромными иконками, сортировкой и контекстным меню.

2. **Работа с архивами:**
   - Создание `.zip` архивов из выбранных файлов и папок с сохранением структуры.
   - Распаковка архивов форматов `.zip`, `.tar`, `.tar.gz`, `.tgz`, `.tar.bz2`.
   - Безопасная распаковка с защитой от путей вида `../../file`.
   - Предварительный просмотр содержимого поддерживаемых архивов без полной распаковки.

3. **Поиск файлов:**
   - Поиск по имени файла или папки в текущей директории и поддиректориях.
   - Поиск текста внутри текстовых файлов.
   - Фильтрация результатов по расширениям.
   - Асинхронное выполнение поиска в скрываемой панели с кнопками остановки и сброса.

## Структура проекта

```text
archive_manager/
├── main.py                    # Главная точка входа в приложение
├── run.bat                    # Быстрый запуск для Windows
├── run.sh                     # Быстрый запуск для Linux/macOS
├── requirements.txt           # Зависимости проекта
├── README.md                  # Документация проекта

    ├── search_utils.py        # Утилиты поиска по имени и содержимому
    └── ui/                    # Компоненты интерфейса PySide6
        ├── action_bar.py      # Верхняя панель действий
        ├── dialogs.py         # Диалог просмотра архива
        ├── icons.py           # Единые монохромные иконки

        ├── theme.py           # Общая тема, отступы и hover-состояния
        └── workers.py         # Фоновые задачи Qt
```

## Установка зависимостей

Перед первым запуском установите зависимости:

```bash
python -m pip install -r requirements.txt
```

Для Linux/macOS, если команда `python` указывает не на Python 3, используйте:

```bash
python3 -m pip install -r requirements.txt
```

## Инструкция по запуску

### Windows

Дважды кликните по файлу:

```bat
run.bat
```

Или выполните в консоли:

```bat
python main.py
```

### Linux / macOS

Выдайте права на исполнение скрипту и запустите его:

```bash
chmod +x run.sh
./run.sh
```

Или запустите напрямую:

```bash
python3 main.py
```

## Планы по доработке

- Drag-and-Drop для перемещения файлов.
- Прогресс-бар для упаковки и распаковки больших архивов.
- Вкладки для одновременной работы с несколькими директориями.
- Избранные папки для быстрого доступа.
- Создание запароленных ZIP-архивов через отдельную библиотеку, например `pyzipper`.
