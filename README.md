# Курсовой проект: Нативный менеджер файлов и архиватор

**Выполнил:** Никогосян Юрий Сасунович  
**Группа:** РИВ-240938у  
**Тема:** Разработка нативного десктопного приложения «Менеджер файлов (Архиватор)»  
**Версия приложения:** 1.4.66  

## Аннотация
Данный проект представляет собой современное кроссплатформенное десктопное приложение для управления файловой системой и работы с архивами, разработанное в рамках курсового проекта. Программа обладает интуитивно понятным графическим интерфейсом и предоставляет все необходимые инструменты для навигации, базовых операций с файлами, создания и распаковки ZIP-архивов, а также расширенного поиска.

## Цели и задачи проекта
- Разработка удобного инструмента для взаимодействия с файловой системой.
- Реализация функций архивации и декомпрессии без использования сторонних GUI-клиентов.
- Применение современных подходов к разработке пользовательских интерфейсов (Qt/PySide6).
- Обеспечение высокой производительности при работе с большим количеством файлов благодаря использованию многопоточности.

## Основные возможности

### Навигация и управление файлами
- Просмотр содержимого директорий в удобном табличном виде с возможностью сортировки.
- Полноценная навигация: переход назад, вперёд (в том числе кнопками мыши), на уровень выше и в домашнюю директорию.
- Переход по введённому пути и выбор директории через встроенный диалог.
- Базовые файловые операции: создание папок, переименование, удаление (с безопасным перемещением в системную корзину ОС).
- Копирование и перемещение файлов с наглядным отображением статуса буфера обмена.
- Вычисление размера выбранных папок.
- Отмена последних поддерживаемых действий (`Ctrl+Z`).

### Работа с архивами
- Создание оптимизированных `.zip` архивов из выбранных файлов и папок.
- Распаковка архивов популярных форматов: `.zip`, `.tar`, `.tar.gz`, `.tgz`, `.tar.bz2`.
- Просмотр содержимого архива без необходимости его предварительной распаковки.
- Защита от извлечения файлов за пределы целевой директории.

### Предпросмотр и поиск
- Информативная правая панель предпросмотра: отображение миниатюр для изображений и первого кадра для видео (без запуска сторонних плееров).
- Открытие файлов в программах по умолчанию по двойному клику.
- Интеллектуальный поиск: по имени файла, по содержимому текстовых файлов, с возможностью фильтрации по расширениям.
- Динамическое обновление текущей папки раз в секунду: изменения извне отображаются без мигания таблицы.

## Пользовательский интерфейс и горячие клавиши

Интерфейс построен по принципам нативного дизайна и включает в себя адаптивную верхнюю панель, строку пути, таблицу файлов, контекстные меню и панель поиска.

| Действие | Сочетание клавиш |
| --- | --- |
| Назад / Вперёд | `Alt+Left` / `Alt+Right` |
| Вверх по иерархии | `Alt+Up` |
| Домашняя папка | `Alt+Home` |
| Обновить список | `F5` |
| Открыть панель поиска | `Ctrl+F` |
| Отменить последнее действие | `Ctrl+Z` |
| Открыть объект | `Enter` |
| Переименовать | `F2` |
| Удалить объекты | `Delete` |

## Технологический стек

Проект реализован с использованием современных технологий и библиотек:
- **Язык программирования:** Python 3.10+
- **Графический фреймворк:** PySide6 (Qt for Python)
- **Работа с медиа:** imageio-ffmpeg (генерация видео-превью), Pillow + pillow-heif (поддержка формата `.heic/.heif`)
- **Системные модули:** pathlib, os, shutil, zipfile, tarfile
- **Многопоточность:** QThreadPool, QRunnable (для выполнения ресурсоемких задач в фоновом режиме)
- **Сборка:** PyInstaller (для упаковки в единый исполняемый файл `.exe`)

## Руководство по запуску и сборке

### Настройка окружения для разработки
Создайте виртуальное окружение и установите необходимые зависимости:
```bat
python -m venv .venv
.venv\Scripts\activate
python -m pip install --upgrade pip
pip install -r requirements.txt
```

Для работы с тестами и сборки EXE:
```bat
pip install -r requirements-dev.txt
```

### Запуск приложения
**Windows:**
```bat
run.bat
```
или напрямую:
```bat
python main.py
```

**Linux / macOS:**
```bash
python3 main.py
```

### Production-сборка (Windows)
Для получения независимого исполняемого файла используйте подготовленный скрипт:
```bat
.\build_exe_onefile.bat
```
Результат сборки (с включенным Windows manifest, версией и иконкой) будет находиться по пути: `dist\ArchiveManager.exe`.

*Примечание:* Бинарники `ffmpeg.exe` не хранятся в репозитории для экономии места; `imageio-ffmpeg` автоматически интегрируется при сборке.

## Архитектура проекта

Проект разделён на контроллеры, UI-компоненты, табличные виджеты и независимые preview-генераторы.
`archive_app/app.py` теперь отвечает за сборку компонентов и связывание сигналов, а бизнес-логика вынесена в отдельные классы.

```text
archive_manager/
├── main.py                         # Точка входа в приложение
├── requirements.txt                # Зависимости
├── requirements-dev.txt            # Зависимости для тестов и сборки
├── build_exe_onefile.bat           # Production-сборка в единый EXE
├── run.bat                         # Быстрый запуск из исходников
├── packaging/windows/              # Manifest и version info для EXE
├── tests/                          # Автоматизированные тесты pytest
└── archive_app/                    # Основной пакет приложения
    ├── app.py                      # Сборка окна и связывание компонентов
    ├── archive_utils.py            # Безопасная работа с архивами
    ├── file_utils.py               # Низкоуровневые файловые операции
    ├── preview_utils.py            # Фасад предпросмотра
    ├── search_utils.py             # Поиск файлов и текста
    ├── controllers/                # Контроллеры бизнес-логики
    │   ├── action_manager.py       # QAction, горячие клавиши и меню действий
    │   ├── clipboard_manager.py    # Буфер копирования/вырезания
    │   ├── file_operations.py      # Удаление, копирование, ZIP, распаковка
    │   └── navigation_manager.py   # История назад/вперёд/вверх/домой
    ├── previewers/                 # Генераторы превью по типам файлов
    │   ├── dispatcher.py           # Выбор нужного previewer
    │   ├── image_preview.py        # Фото, HEIC/HEIF
    │   ├── video_preview.py        # Видео-кадры через ffmpeg/imageio-ffmpeg
    │   ├── document_preview.py     # Документы и PDF
    │   ├── audio_preview.py        # Аудио и обложки
    │   └── common.py               # Общие типы и форматирование
    └── ui/                         # Виджеты интерфейса
        ├── action_bar.py           # Адаптивная панель действий
        ├── dialogs.py              # Диалоги
        ├── file_table.py           # Таблица файлов
        ├── search_table.py         # Таблица результатов поиска
        ├── table_delegates.py      # Делегаты и стили отрисовки таблиц
        ├── tables.py               # Совместимый фасад импортов таблиц
        ├── window_layout.py        # Сборка UI главного окна
        ├── navigation_bar.py       # Навигационная панель
        ├── preview_panel.py        # Панель предпросмотра
        ├── search_panel.py         # Панель поиска
        ├── theme.py                # QSS-стили
        └── workers.py              # Фоновые задачи
```
